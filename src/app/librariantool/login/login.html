The Librarian Tool helps you manage your subscription to databases offered by
Phoenix Bioinformatics. You can use the Librarian Tool to:

- View subscriptions
- Manage your institution's IP address ranges
- View and request usage report
New feature will be added as we continue to develop this tool.

Access to the Tool is given during account setup. If you need to request access
to the Librarian Tool, please contact Phoenix Bioinformatics.
