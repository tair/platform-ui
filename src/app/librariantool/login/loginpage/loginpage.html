The Admin Portal helps you manage your subscription to databases offered by
Phoenix Bioinformatics. You can use the Admin Portal to:

View subscriptions
Manage your institution's IP address ranges
Request usage report
New feature will be added as we continue to develop this portal.

Access to the portal is given during account setup. If you need to request access
to the Admin Portal, please contact Phoenix Bioinformatics.
