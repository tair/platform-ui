Partner API

A partner is a scientific data or informatics system that comprises a unit for subscription, authentication, and authorization. The user accesses the partner system through a well-defined set of URIs identified by a set of regular-expression patterns. Partners have individual subscription terms for display on subscription pages. Terms have period, price, and group discount percentage. Partners have regular expression patterns that identify the complete set of URIs for the partner, or alternatively identify the partner from a URI. Partners have subscriptions by parties. Parties have IP counts and limit values. Partners also own access rules (combinations of URI patterns and access types.
